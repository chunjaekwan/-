안녕하세요 경기도 일산에서 살고있는
웹 퍼블리셔 지망생 천재관 입니다.
웹디자인과 프론트엔드에 열정을 가지고
미래를 꿈꾸며 성장하고 싶습니다.

  • HTML

    • 구성 요소들의 의미와 사용 방식을 해석 및 판단하여
    그에 맞도록 문서를 구조화하여 작성 가능합니다.
    또는 개발자 도구를 이용해서 웹 개발의 효율성을
    극대화 할 수 있습니다.

  • CSS

    • 마크업된 문서의 디자인과 레이아웃 만들고
    인터렉티브한 홈페이지 제작을 할 수 있습니다.

  • SASS

    • CSS를 능숙하게 다루며 협업능력과 생산성을 향상
    시킬 수 있습니다.

  • Javascript

    • 사용자와 상호작용하는 UI 구현 및 서버를 API를
    활용할 수 있습니다.

    Figma

피그마를 활용하여 UI설계 및 디자인이 가능 합니다.
